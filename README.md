# intellitracker

Gym Tracker Mobile App

A user-friendly mobile application that helps users track their daily gym workouts and visualize their fitness progress through comprehensive summaries and interactive graphs.

Overview

The Gym Tracker App is designed to help users log workouts, monitor progress, and stay consistent with their fitness goals. It provides an intuitive interface for recording exercises, sets, and reps, along with visual insights to track improvement over time.

This application follows an API-first approach, making it scalable for future web dashboards, admin panels, or integrations.
